CloudERP Simplebooks

A minimal offline-friendly bookkeeping API with manual double-entry controls.

API Endpoints

The API listens on http://localhost:5010 by default.

  • GET /health — service status
  • POST /customers, POST /vendors — master data
  • POST /sales-orders, POST /sales-orders/:id/confirm, POST /sales-orders/:id/to-invoice
  • POST /invoices, POST /invoices/:id/issue, POST /invoices/:id/record-payment
  • POST /bills, POST /bills/:id/approve, POST /bills/:id/record-payment
  • GET /reports/trial-balance, GET /reports/ar-aging, GET /reports/ap-aging

Getting Started

  1. Run npm start to launch the API server.
  2. Use curl or an HTTP client to interact with the endpoints.
  3. Run npm test to execute the accounting flow scenarios.
